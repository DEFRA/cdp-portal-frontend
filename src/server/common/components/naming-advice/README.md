# Naming Advice

## Display

The component shows:

1. **Good examples** - Valid names using the provided categories (e.g., `seed-vault-frontend`)
2. **Bad examples** - Names to avoid (phases, versions, temp words)
3. **Documentation link** - Link to naming conventions guide

## Notes

- Used in create forms to guide users on naming
- Links to the full naming conventions documentation
- Shows tick/error icons for good/bad examples
