# Warning

## Accessibility

- Uses `role="alert"` for screen reader announcement
- `tabindex="-1"` allows programmatic focus

## CSS Classes

| Class                  | Description           |
| ---------------------- | --------------------- |
| `app-warning`          | Base warning class    |
| `app-warning--inverse` | Dark background style |

## Notes

- Content priority: caller block > html > text
- Icon uses `appWarningIcon` component
- Designed for important alerts that need user attention
